# Stellaris mod: Star Seekers Notifications

**English** | [Русский](README.ru.md)

The mod provides advance election notifications, shows the estimated Unity cost of influencing an election based on empire size, and estimates how much Unity will have been accumulated by the time the election begins.

The mod takes the current form of government into account and detects when it changes.

Settings are available through a dedicated menu under **Edicts**. There, you can choose whether to receive notifications 12, 6, or 3 months before an election begins.
